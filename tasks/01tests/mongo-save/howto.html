01tests

mongo-save

cmd: $ node start 01tests/mongo-save

description:

The example shows how to use dex8lib.Mongo library.



input.js

Contains object which should be inserted into mongo database.



functions:
  • mongoConnect.js - connects to mongo database and creates mongoose model
  • modifyInput.js - modifies object defined in input.js
  • mongoSave.js - saves modified input object into mongodb


main.js
  1. execute mongoConnect, modifyInput and mongoSave serially
  2. repeat mongoSave 3 times
